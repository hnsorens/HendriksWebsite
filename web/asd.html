Core Features

Process Management

  • Preemptive round-robin scheduler
  • Copy-on-write fork() implementation
  • Process isolation with ring transitions
  • PID hash table for O(1) lookups

Memory Management

  • Memory-mapped files

Filesystem

  • Virtual Filesystem (VFS) layer
  • File descriptor management
  • Block device caching

System Calls

  • 21 POSIX-compatible syscalls
  • SYSCALL/SYSRET instruction usage
  • Process isolation enforcement
  • Signal delivery system

Technical Highlights

Context Switching

Implemented efficient process switching with TSS and custom assembly routines:

; x86-64 context switch
switch_context:
    push rbp, rbx, r12-r15      ; Save callee-saved registers
    mov [rdi], rsp              ; Save current stack pointer
    mov rsp, [rsi]              ; Load new stack pointer
    pop r15, r14, r13, r12, rbx, rbp
    ret

Virtual Memory

4-level paging implementation with page fault handling:

[MM] Mapped 0x400000-0x600000 to physical 0x100000 (2MB page)
[MM] Page fault at 0x7ffff000: COW handling for process 42
[MM] 4KB page allocated at 0x7ffff000
